Desarrollo Web

Esta es una página de práctica creada como parte de una actividad asignada por el profesor de la materia de Software. El objetivo de este ejercicio es repasar y reforzar nuestros conocimientos en desarrollo web utilizando HTML y CSS. A través de esta práctica, ponemos en uso lo aprendido en clase para diseñar y estructurar páginas web de manera básica, enfocándonos en la correcta aplicación del código y el estilo visual.
